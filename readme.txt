== Description ==

A simple plugin that allows you to download all installed plugin status in CSV file record. It will download all the installed plugin list in CSV file format.

You can also see all the plugins name, date of download file and plugin status either it is active or inactive. It will help you get records of all the plugins status while you change their status.

The CSV file records contain below information :

- Sr. No - A serial number is a unique identifier assigned sequentially to a plugin.

- Plugin Name - The name of the plugin.

- Version - The version of the plugin.

- Status - The status(active/deactive) of the plugin.

- Time - Time of the records export.
